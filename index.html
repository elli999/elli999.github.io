三种模式（记忆！）

三种常见的模式：

  • Normal Mode
  • Insert Mode
  • Visual Mode

Normal mode和Insert Mode的切换有Append，Insert，Replace，Subsitute 记作AIRS，这种记忆方法特别方便，英文单词的意思是空气。

Hexo+Markdown+Vim 打造无敌笔记系统

Posted on 2020-06-10 Edited on 2020-07-27 In Software , 笔记方法学 , VIM

本文是本人呕心沥血之作，望能对大家有所帮助！

简介

使用Hexo + Vim + Markdown打造笔记系统

Hexo是一个用户量众多的静态博客系统，使用Markdown语法进行博客编写，而Vim是一个经典的编辑器，在熟练掌握三者之后进行笔记（博客）编写会如丝般顺滑，这里把Hexo博客系统来作为笔记系统使用，并将其部署在网上，我们可以随时通过任何联网设备查看笔记。（TODO：有空补一个演示视频）

本人曾在Windows和Ubuntu均成功搭建此系统。

由于Vim学习成本过高，如果不想折腾的朋友可使用Hexo + Visual Studio Code（搭配Markdown插件）使用。

本文只记录搭建笔记系统相关知识，Hexo、Vim、Markdown知识请查阅《Hexo笔记》、《Vim笔记》、《Markdown笔记》

我也在不断学习和不断摸索这套系统，如果大家有问题欢迎咨询联系我，我愿意共同探讨解决问题，联系方式在首页 -> about页面。
